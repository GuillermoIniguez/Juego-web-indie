# Juego Web Básico

Este es un juego web simple donde el jugador debe hacer clic en un cuadro para ganar puntos. El cuadro se mueve a una nueva posición cada vez que se hace clic en él.

## Cómo jugar

1. Abre el archivo `index.html` en tu navegador.
2. Haz clic en el cuadro rojo.
3. Cada clic suma un punto y el cuadro se mueve a una nueva posición.
4. Tu puntaje se mostrará en pantalla.

## Tecnologías usadas

- HTML
- CSS
- JavaScript
